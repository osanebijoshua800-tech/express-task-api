\# Express Task API



A simple backend application built with Node.js and Express to handle task management. This project was built to learn the fundamentals of REST APIs, handling HTTP requests, and testing with Postman.



\## Features

\- \*\*GET /\*\* - A simple health check route that returns a welcome message.

\- \*\*POST /tasks\*\* - Accepts a JSON body containing a task `title` and `status`, processes the data, and returns a `201 Created` success response.
